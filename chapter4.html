Multiple Choice Questions (MCQ)

MCQ
1. In a combinational logic circuit, the output depends on:
A) The current input values only
B) Previous input values
C) Feedback from outputs
D) Clock signal
Answer: A) The current input values only
Combinational circuits have no memory or feedback; output is a function of present inputs only.
MCQ
2. Which of the following is NOT a combinational logic circuit?
A) Half adder
B) Multiplexer
C) Flip-flop
D) Encoder
Answer: C) Flip-flop
Flip-flop is a sequential circuit with memory; others are purely combinational.
MCQ
3. What is the main difference between analysis and design of combinational circuits?
A) Analysis starts from a circuit, design starts from a function or specification
B) Analysis gives a truth table, design gives a Boolean expression
C) Analysis uses K-maps, design does not
D) There is no difference
Answer: A) Analysis starts from a circuit, design starts from a function or specification
In analysis, you determine the function; in design, you create the circuit from requirements.
MCQ
4. Which combination of logic gates can implement any combinational circuit?
A) Only AND gates
B) Only OR gates
C) AND, OR, NOT gates
D) Only XOR gates
Answer: C) AND, OR, NOT gates
These three gates form a complete set for Boolean logic.
MCQ
5. What are the outputs of a half adder?
A) Sum and Carry
B) Product and Sum
C) XOR and XNOR
D) Sum only
Answer: A) Sum and Carry
Half adder adds two bits and produces sum and carry outputs.
MCQ
6. Which is the correct Boolean expression for the sum output of a full adder?
A) xy + yz + xz
B) x XOR y XOR z
C) x AND y AND z
D) x + y + z
Answer: B) x XOR y XOR z
Sum = x ⊕ y ⊕ z in a full adder.
MCQ
7. A 3-to-8 line decoder has:
A) 3 inputs, 8 outputs
B) 8 inputs, 3 outputs
C) 3 inputs, 3 outputs
D) 8 inputs, 8 outputs
Answer: A) 3 inputs, 8 outputs
A 3-to-8 decoder produces 8 outputs, one for each input combination.
MCQ
8. What does a multiplexer (MUX) do?
A) Encodes data
B) Selects one of many inputs to pass to the output
C) Adds binary numbers
D) Decodes data
Answer: B) Selects one of many inputs to pass to the output
A MUX uses select lines to choose which input appears at the output.
MCQ
9. The output of a 2-to-4 decoder is high for:
A) All combinations
B) No combinations
C) Exactly one input combination
D) Two input combinations
Answer: C) Exactly one input combination
Only one output line is high for any input combination in a decoder.
MCQ
10. What is the main purpose of an encoder?
A) To add binary numbers
B) To select one input
C) To convert many inputs into a coded output
D) To decode binary data
Answer: C) To convert many inputs into a coded output
An encoder translates active input lines into a binary code.

True or False (TF)

TF
11. Combinational circuits can have feedback loops.
Answer: False
Combinational circuits have no feedback; only sequential circuits do.
TF
12. The output of a combinational circuit changes immediately when the input changes.
Answer: False
There is always a propagation delay due to gate switching.
TF
13. The sum output of a half adder is an XOR function of the inputs.
Answer: True
Sum = A ⊕ B for a half adder.
TF
14. A 4-to-16 decoder has 4 outputs.
Answer: False
It has 16 outputs, one for each input combination.
TF
15. The carry output of a full adder is high when at least two inputs are high.
Answer: True
Carry = xy + yz + xz: any two or all three inputs high produces carry out.
TF
16. In a 4-to-1 multiplexer, there are 4 select lines.
Answer: False
A 4-to-1 MUX has 2 select lines (since 2² = 4 inputs).
TF
17. Encoders and decoders are inverse operations of each other.
Answer: True
An encoder converts from many lines to a code; a decoder does the reverse.
TF
18. A demultiplexer routes a single input to one of several outputs, depending on select lines.
Answer: True
DEMUX is the opposite of MUX, distributing data to one output at a time.
TF
19. In a seven-segment display, each segment is controlled by a separate output from the decoder.
Answer: True
Each segment (a–g) is controlled by its own logic output.
TF
20. Tri-state gates can output a logic high, logic low, or a high-impedance (Hi-Z) state.
Answer: True
Tri-state gates are used for bus systems and allow "disconnecting" from the bus.

Short Answer Questions

Short
21. What is a truth table and why is it important in combinational logic design?
Answer: A truth table lists all possible input combinations and their corresponding outputs. It is essential for analyzing and designing logic circuits.
It provides a systematic way to specify circuit behavior and derive logic expressions.
Short
22. Write the Boolean function for the output “S” of a full adder with inputs x, y, z.
Answer: S = x ⊕ y ⊕ z
Sum output is the XOR of all three inputs.
Short
23. How many select lines are needed for an 8-to-1 multiplexer?
Answer: 3
2³ = 8, so 3 select lines are required to address 8 inputs.
Short
24. In a BCD-to-Excess-3 code converter, what is the output for input 0101 (BCD for 5)?
Answer: 1000
BCD 5 is 0101, Excess-3 = 5+3 = 8 → 1000.
Short
25. Give an example of a real-world application for a multiplexer.
Answer: Data selection in communication systems, such as choosing between multiple data sources to send over one line.
MUXs are used extensively in digital systems to select data sources.
Short
26. What is the function of a decoder with enable input?
Answer: The decoder outputs are active only when enable is high (or low, depending on design); otherwise, all outputs are inactive.
Enable input acts as a control to turn the decoder on or off.
Short
27. Why are "don't care" conditions useful when designing combinational circuits?
Answer: They allow for more flexible simplification, resulting in simpler logic expressions and circuits.
Don't cares help in K-map grouping, reducing hardware.
Short
28. What is the advantage of using a tri-state buffer in a bus system?
Answer: It allows multiple devices to share a data bus without interfering with each other by placing outputs in a high-impedance state.
Tri-state buffers prevent contention and allow only one output at a time.
Short
29. Draw the block diagram of a 2-to-4 line decoder and label all inputs and outputs.
Answer: Block with 2 inputs (I1, I0), 4 outputs (Y3, Y2, Y1, Y0). Each output is high for one input combination.
Drawing not shown here; see textbook for standard symbol.
Short
30. What is the output of a 4-to-1 multiplexer if select lines S1=1, S0=0 and inputs I0=0, I1=1, I2=1, I3=0?
Answer: 1
S1=1, S0=0 selects input I2, which is 1.
