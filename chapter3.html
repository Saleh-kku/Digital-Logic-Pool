Logic Simplification – Key Question Pool

Total Questions: 30

Multiple Choice Questions (MCQ)

MCQ
1. What is the primary goal of logic simplification?
A) Increase the number of gates
B) Use the most variables
C) Use the fewest gates possible
D) Make the circuit slower
Answer: C) Use the fewest gates possible
Simplification aims to reduce hardware cost and increase efficiency.
MCQ
2. Which Boolean law allows you to combine AB + AB into AB?
A) Distributive
B) Idempotent
C) Absorption
D) Complementation
Answer: B) Idempotent
Idempotent law: X + X = X, XX = X, so AB + AB = AB.
MCQ
3. In the expression AB + AC + B, which simplification law is used to reduce B + BC to B?
A) Absorption
B) Distributive
C) Demorgan’s
D) Consensus
Answer: A) Absorption
Absorption law: X + XY = X.
MCQ
4. Which of the following best describes a standard SOP form?
A) Each product term may have any number of variables
B) Every product term contains all variables, true or complemented
C) All terms are ORed together
D) All variables are complemented
Answer: B) Every product term contains all variables, true or complemented
Standard SOP ensures all variables appear in every term.
MCQ
5. What is the minimum number of cells in a group on a K-map?
A) 1
B) 2
C) 4
D) 8
Answer: A) 1
A group can be just one cell (if no simplification is possible).
MCQ
6. Which of the following is NOT a valid grouping size in a 4-variable K-map?
A) 1
B) 3
C) 4
D) 8
Answer: B) 3
Groups must be powers of 2: 1, 2, 4, 8, 16.
MCQ
7. How many minterms are there for a three-variable function?
A) 3
B) 6
C) 8
D) 16
Answer: C) 8
2ⁿ minterms for n variables; here, 2³ = 8.
MCQ
8. What does a “don’t care” condition mean in K-map simplification?
A) An input must always be 0
B) The input is always 1
C) The output can be either 0 or 1 for that input combination
D) The variable is missing
Answer: C) The output can be either 0 or 1 for that input combination
"Don't care" cells are used to maximize simplification.
MCQ
9. What is the binary representation of the product term A'BC for variables A, B, C?
A) 100
B) 010
C) 011
D) 101
Answer: C) 011
A' = 0, B = 1, C = 1, so 011.
MCQ
10. In a standard POS form, what does each sum term represent?
A) A minterm
B) A maxterm
C) An input
D) An output
Answer: B) A maxterm
Each sum term in POS represents a maxterm: a sum of all variables in true/complemented form.

True or False (TF)

TF
11. Any Boolean function can be converted into SOP or POS form.
Answer: True
Standardization is always possible for systematic analysis.
TF
12. A Karnaugh map for four variables has 16 cells.
Answer: True
2⁴ = 16; each cell represents a minterm.
TF
13. In K-map simplification, groups must always be rectangular and powers of 2.
Answer: True
Groups: 1, 2, 4, 8, etc., and must be adjacent cells.
TF
14. In a 2-variable K-map, adjacent cells differ by only one variable.
Answer: True
Cells are arranged in Gray code order, so only one variable changes.
TF
15. A group of four 1s in a K-map always results in a three-variable product term.
Answer: False
For a four-variable map, a group of 4 cells eliminates two variables, resulting in a two-variable term.
TF
16. The minimum SOP expression from a K-map is always unique.
Answer: False
Different groupings can sometimes yield different but equally minimal expressions.
TF
17. The consensus theorem is used to eliminate redundant terms in Boolean expressions.
Answer: True
Consensus theorem helps further minimize expressions.
TF
18. In a truth table, an SOP expression is ‘1’ for each input combination where at least one product term is ‘1’.
Answer: True
That is the definition of SOP and how it’s mapped to the truth table.
TF
19. "Don't care" conditions cannot be used for simplification in K-maps.
Answer: False
They are included to help create larger groups and achieve simpler expressions.
TF
20. The expression (A+B)(A+C) can be expanded to SOP form as A + BC.
Answer: False
(A+B)(A+C) = AA + AC + BA + BC = A + AC + AB + BC = A + AB + AC + BC (A + AB = A, A + AC = A), but not A + BC directly.

Short Answer Questions

Short
21. Describe the steps to convert a non-standard product term to standard SOP form.
Answer: Multiply the term by the sum of each missing variable and its complement, doubling the number of terms each time, until all terms include all variables.
This ensures every product term has all variables, in either true or complemented form.
Short
22. What does each cell in a K-map represent?
Answer: Each cell represents a minterm, which is a unique combination of input variable values.
This allows mapping Boolean expressions and truth tables onto the K-map.
Short
23. Write the SOP expression for the output being 1 for input combinations 011 and 101 (for A, B, C).
Answer: A'B C + AB'C
011: A'B C; 101: AB'C.
Short
24. What is the advantage of using “don’t care” conditions in K-map simplification?
Answer: They allow for larger groups, leading to simpler logic expressions and fewer gates.
This increases efficiency in circuit design.
Short
25. What is the difference between an SOP and a POS expression?
Answer: SOP is a sum (OR) of products (AND terms); POS is a product (AND) of sums (OR terms).
Each has its own standard form and application in circuit design and analysis.
Short
26. Give an example of a Boolean expression and its simplified form using K-map or Boolean algebra.
Answer: Example: AB + AB' = A
By applying the distributive law: AB + AB' = A(B + B') = A(1) = A.
Short
27. How many 1s must be included in every group on a K-map?
Answer: Groups must contain 1, 2, 4, 8, etc. (powers of 2) 1s.
Grouping must follow the powers-of-two rule for proper simplification.
Short
28. Write the binary representation for the maxterm A+B'+C for variables A, B, C.
Answer: 101
A=1, B'=0, C=1.
Short
29. Explain the process of constructing a truth table from a standard SOP expression.
Answer: List all combinations of variables; for each, check if any product term is 1 for that input, then output is 1; otherwise, output is 0.
Direct mapping is possible from standard SOP to truth table.
Short
30. Describe the steps to design a combinational circuit from a problem statement.
Answer: 1) Derive the truth table from the problem; 2) Write the unsimplified logic expression; 3) Simplify the expression; 4) Draw the logic circuit from the simplified expression.
This is a systematic approach to digital circuit design.
