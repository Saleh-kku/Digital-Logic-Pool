Homework 1: Number Systems & Binary Codes

Instructions: Answer all questions and show all your steps/work for full credit.

  1. Convert the following decimal numbers to binary, octal, and hexadecimal:
    • a) 45
    • b) 123
  2. Convert the following binary numbers to decimal:
    • a) 110101
    • b) 1011101
  3. Express the decimal number 98 in BCD (Binary Coded Decimal).
  4. What is the Gray code equivalent of the binary number 1011?
  5. Explain the main difference between analog and digital signals. Give a real-world example for each.
  6. Find the 2’s complement of the following 8-bit binary numbers:
    • a) 00110101
    • b) 11100001
  7. How many different values can be represented using 5 bits?
  8. What is the hexadecimal representation of the binary number 11011010?
  9. List the digits used in the octal number system and explain its base.
  10. Convert the following octal numbers to decimal:
    • a) 54
    • b) 137

Homework 2: Boolean Algebra

Instructions: Answer all questions. Show simplification steps where required.

  1. Simplify the Boolean expression using algebraic methods:
    F = AB + A(B + C) + B(B + C)
  2. State and briefly explain the commutative, associative, and distributive laws of Boolean algebra with examples.
  3. Express the following Boolean function in canonical (sum of minterms) form:
    F(A, B, C) = A'B + AC
  4. Verify DeMorgan’s Theorem for the expression:
    (A + B)'
  5. Write the dual of the Boolean expression: XY + X'Z
  6. Draw the truth table for the function: F = AB' + A'B
  7. List the basic identities of Boolean algebra.
  8. Define minterm and maxterm. Give one example of each for variables X and Y.
  9. Convert the following logic circuit (describe verbally or draw) into a Boolean expression:
    An AND gate with inputs A and B, whose output goes to an OR gate along with input C.
  10. Simplify the following using Boolean algebra: F = (A + B)(A + C)

Homework 3: Logic Simplification

Instructions: Answer all questions. Show all work and simplification steps.

  1. Using Boolean algebra, simplify the following expression:
    F = AB + AC + B
  2. Convert the following non-standard SOP expression into standard SOP form:
    F = AC + B
  3. For the Boolean function F(A,B,C) = Σ(1,2,6,7):
    • a) Draw the truth table.
    • b) Simplify using a Karnaugh map (K-map).
  4. What is a "don't care" condition? How can it help in logic simplification?
  5. Write the standard POS (Product of Sums) form for the function that is 0 for minterms 0 and 3 (for variables A and B).
  6. Explain the steps to convert a truth table to a standard SOP expression.
  7. Given the function F(A, B, C) = AB' + BC, implement it using only NAND gates.
  8. Explain how a standard product term maps to a single cell in a K-map.
  9. Simplify the following Boolean expression using K-map:
    F(A,B,C) = Σ(1,3,5,7)
  10. Give an example of a logic function with at least 3 variables and show both its truth table and minimized expression.

Homework 4: Combinational Logic Circuits

Instructions: Answer all questions. Include diagrams where appropriate.

  1. What is a combinational circuit? List three differences between combinational and sequential circuits.
  2. Given the truth table for a 2-input AND gate, write the Boolean expression and draw the logic circuit.
  3. Design a 3-bit binary to Excess-3 code converter. Provide the truth table and the Boolean expressions for each output bit.
  4. Explain the operation of a half adder. Draw its circuit diagram and write the truth table.
  5. What is the function of a multiplexer? Draw the block diagram of a 4-to-1 multiplexer and explain its operation.
  6. How many select lines are required for a 16-to-1 multiplexer? Justify your answer.
  7. Write the Boolean equation for the sum (S) and carry (C) outputs of a full adder.
  8. Describe how a 3-to-8 line decoder works. Give an example of its application.
  9. Explain the difference between an encoder and a decoder. Give one practical example of each.
  10. What is a tri-state buffer? Where is it used in digital systems?
